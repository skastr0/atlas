# Support

atlas is an experimental, solo-maintained project. There is no guaranteed support response time.

Support is limited to reproducible bugs, documentation corrections, and scoped proposals that fit the project goals.

Support does not include private consulting, urgent response, custom feature implementation, or debugging unsupported forks, private knowledge bases, credentials, or infrastructure outside this repository.

Use GitHub issues for reproducible bugs and scoped proposals. Please include the version or commit, operating system, command run, expected behavior, actual behavior, and any relevant logs with secrets removed.

Report suspected vulnerabilities through the private process in SECURITY.md, not public issues.
